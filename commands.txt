/help - Справка
/google - бот будет гуглить перед ответом
/bard - запрос к Google Bard
/sum - делает выжимку из текста по ссылке (сумморизацию текста) /sum URL
/sum2 - то же что и /sum но не берет ответы из кеша, повторяет запрос заново
/activate - административная команда для активации работы бота в этой теме
/deactivate - административная команда для деактивации работы бота в этой теме
/id - административная команда покажет id юзера и группы
/add - административная команда для добавления стоп слова
/del - административная команда для удаления стоп слова
/export - административная команда для экспорта данных
